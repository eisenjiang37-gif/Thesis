\noindent \textbf{1. 文献综述和开题报告要求：}
\begin{enumerate}[nosep, leftmargin=2em] % nosep 消除列表项间的额外间距，使其完全匹配 18bp 行距
    \item 查阅国内外相关文献资料10篇以上（其中外文文献不少于5篇），近 5 年文献不少于 3 篇。在此基础上完成文献综述，正文要求4000字以上。
    \item 完成外文翻译，正文（中文）要求3500字以上。
    \item 完成开题报告，正文要求4500字以上。
\end{enumerate}

\vspace{18bp} % 段落间的精准过渡

\noindent \textbf{2. 文献综述要求：}
\begin{enumerate}[nosep, leftmargin=2em]
    \item 掌握多普勒信号的特征与处理方法，了解其在生物探测领域的应用价值、技术优势与局限，以及国内外研究进展。
    \item 综述现有手势信号特征提取与分类算法，分析其国内外研究现状。
    \item 对比各类手势识别技术，阐明采用多普勒雷达实现手势识别的研究意义。
    \item 做好毕业论文相关知识的准备工作。
\end{enumerate}

\vspace{18bp}

\noindent \textbf{3. 开题报告要求：}
\begin{enumerate}[nosep, leftmargin=2em]
    \item 做好毕业论文相关知识的准备工作。
    \item 给出基于多普勒雷达的手势识别系统设计方案，选定核心处理算法。
    \item 对论文要完成的工作有清晰的认识，给出具体的研究方案，预见可能遇到的问题和困难。
    \item 提出论文的研究成果或预期达到的目标。
    \item 安排好论文的进度，对于每个阶段的工作有详细的规划。
    \item 给出本人已进行的研究或设计工作及已取得的结果。
\end{enumerate}
