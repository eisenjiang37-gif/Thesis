%----------正文----------

以5.8GHz三通道TDM-SIMO连续波多普勒雷达为硬件平台，基于米氏散射理论对手掌运动的电磁散射特性进行建模，提取具有物理意义的确定性多普勒相位特征，实现非接触式手势识别系统的设计、搭建与评估。具体任务要求如下：

（1）研究多普勒雷达近场散射理论，建立手掌运动与各通道接收信号间的确定性相位调制关系；

（2）搭建完整硬件采集链路，实现高精度、高稳定性的TDM-SIMO雷达数据采集系统；

（3）设计并实现三种手势分类方案：确定性信号特征+SVM、频谱图+CNN迁移学习、原始时序+LSTM；

（4）针对连续交互场景中缺乏静止段的问题，提出鲁棒性更强的特征提取方法；

（5）完成多折交叉验证实验，对三种方案在准确率、训练时间与资源消耗等维度进行系统对比。
